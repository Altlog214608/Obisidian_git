# 검사 및 훈련 시나리오
tags: #검사 #훈련 #향기검사

## 🧪 검사 종류

- 역치 검사 (Threshold)
- 식별 검사 (Discrimination)
- 인지 검사 (Identification)

## 🎯 검사 시나리오 공통 흐름

1. Start → WaitReady → Proceed
2. TryScent → 응답선택 → Confirm
3. 결과 저장 → 점수 계산 → 엑셀 출력

## 🧠 훈련 흐름

- `uiTrainST*`, `sequentialTrainST`, `rateTrainST`
- `TrainID`도 유사 흐름 구조
